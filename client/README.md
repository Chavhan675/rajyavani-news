# Rajyavani Marathi News Portal

Rajyavani is a professional Marathi news website built using:

- Next.js (Frontend)
- Node.js + Express (Backend)
- MongoDB (Database)

---

## Features

- Marathi News Homepage
- Breaking News
- Category News
- News Details Page
- Search News
- User Login / Signup
- Submit News
- Admin Panel
- Earnings System
- Trending News
- Featured News

---

## Project Structure

Rajyavani
│
├── client (Next.js Frontend)
│
├── server (Node.js Backend)
│
└── uploads (News Images)
